# 📦 Полная инструкция: Перенос проекта с Windows на macOS

## 📚 Какие файлы созданы для вас:

1. **`ПЕРЕНОС_НА_MACOS.md`** - Детальная инструкция по копированию файлов с Windows на флешку
2. **`RUN_MACOS.md`** - Полная инструкция по установке и запуску на macOS
3. **`QUICK_START_MACOS.md`** - Быстрый старт (мини-инструкция)
4. **`ИНСТРУКЦИЯ_ПЕРЕНОС_WINDOWS_TO_MAC.md`** - Этот файл (общая сводка)

---

## 🎯 Пошаговый план действий

### НА WINDOWS:

#### Шаг 1: Подготовка файлов для копирования

**Что КОПИРОВАТЬ:**
- ✅ Вся папка `backend/` (кроме `__pycache__`)
- ✅ Вся папка `frontend/` (кроме `node_modules` и `dist`)
- ✅ Папки: `templates/`, `examples/`, `uploads/`
- ✅ Файлы: `requirements.txt`, `setup.sh`, `run_backend.sh`, `run_frontend.sh`
- ✅ Все файлы `.md` (документация)

**Что НЕ копировать:**
- ❌ `.venv/` - виртуальное окружение
- ❌ `__pycache__/` - кэш Python
- ❌ `node_modules/` - зависимости Node.js
- ❌ `*.db` - базы данных (опционально, если нужны данные)
- ❌ `*.log` - логи

#### Шаг 2: Копирование на флешку

1. Подключите флешку к Windows
2. Откройте проводник
3. Скопируйте папку `MailSenderZilla` на флешку
4. **Безопасно извлеките** флешку (правой кнопкой → Извлечь)

---

### НА macOS:

#### Шаг 1: Скопируйте проект с флешки

1. Подключите флешку к Mac
2. Откройте Finder
3. Найдите флешку в боковом меню
4. Скопируйте папку `MailSenderZilla` в `~/Documents/` (Документы)

#### Шаг 2: Установка

Откройте **Terminal** и выполните:

```bash
# Перейдите в проект
cd ~/Documents/MailSenderZilla

# Дайте права на выполнение скриптам
chmod +x setup.sh run_backend.sh run_frontend.sh

# Запустите установку
./setup.sh
```

Скрипт автоматически установит всё необходимое!

#### Шаг 3: Запуск

**Нужно 2 окна Terminal:**

**Terminal 1 - Backend:**
```bash
cd ~/Documents/MailSenderZilla
./run_backend.sh
```

**Terminal 2 - Frontend:**
```bash
cd ~/Documents/MailSenderZilla
./run_frontend.sh
```

#### Шаг 4: Откройте браузер

Перейдите на: **http://localhost:3000**

---

## ⚠️ Требования на macOS

Перед установкой убедитесь, что установлены:

1. **Python 3.8+**
   ```bash
   python3 --version
   ```
   Если нет: `brew install python3`

2. **Node.js 16+**
   ```bash
   node --version
   ```
   Если нет: `brew install node`
