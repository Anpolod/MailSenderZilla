# Установка и запуск MailSenderZilla

## ✅ Что уже сделано

1. ✅ **Виртуальное окружение Python создано** (`.venv/`)
2. ✅ **Все Python зависимости установлены** в виртуальное окружение
3. ✅ **Node.js установлен** (v25.2.1)
4. ✅ **Все Node.js зависимости установлены** в `frontend/node_modules/`
5. ✅ **База данных инициализирована** (`Main_DataBase.db`)

Все установки изолированы в виртуальных пространствах!

## 🚀 Запуск приложения

### Вариант 1: Использовать готовые скрипты

**Backend (в одном терминале):**
```bash
./run_backend.sh
```

**Frontend (в другом терминале):**
```bash
./run_frontend.sh
```

### Вариант 2: Ручной запуск

**Backend:**
```bash
source .venv/bin/activate
python -m backend.app
```

**Frontend:**
```bash
cd frontend
npm run dev
```

## 📍 Адреса

- **Backend:** http://localhost:5000
- **Frontend:** http://localhost:3000

## 📁 Структура виртуальных пространств

```
MailSenderZilla/
├── .venv/                    # Python виртуальное окружение
│   ├── bin/                  # Python интерпретатор и скрипты
│   └── lib/                  # Все Python пакеты (изолированы)
├── frontend/
│   └── node_modules/         # Все Node.js пакеты (изолированы)
└── Main_DataBase.db          # База данных SQLite
```

## 🔄 Переустановка

Если нужно переустановить все зависимости:

```bash
# Удалить виртуальное окружение
rm -rf .venv

# Удалить Node.js зависимости
rm -rf frontend/node_modules

# Переустановить все
./setup.sh
```

## ⚙️ Настройка

Создайте файл `.env` для настройки (опционально):

```bash
SECRET_KEY=your-secret-key
TELEGRAM_BOT_TOKEN=your-bot-token
TELEGRAM_CHAT_ID=your-chat-id
```

## 📝 Примечания

- Все Python пакеты установлены в `.venv/` - не затрагивают системные установки
- Все Node.js пакеты установлены в `frontend/node_modules/` - не затрагивают глобальные пакеты
- База данных создана автоматически при первом запуске миграции
