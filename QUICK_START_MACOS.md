# ⚡ Быстрый старт на macOS

## 📋 Чеклист перед запуском

- [ ] Python 3.8+ установлен (`python3 --version`)
- [ ] Node.js 16+ установлен (`node --version`)
- [ ] Проект скопирован с флешки на Mac
- [ ] Открыт Terminal

---

## 🚀 Быстрая установка (3 команды)

```bash
# 1. Перейдите в проект
cd ~/Documents/MailSenderZilla

# 2. Дайте права на выполнение
chmod +x *.sh

# 3. Запустите установку
./setup.sh
```

---

## ▶️ Запуск (2 терминала)

### Terminal 1 - Backend:
```bash
cd ~/Documents/MailSenderZilla
./run_backend.sh
```

### Terminal 2 - Frontend:
```bash
cd ~/Documents/MailSenderZilla
./run_frontend.sh
```
